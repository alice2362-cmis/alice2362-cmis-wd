  4. Edit the html file created however you like, with all the correct properties like head, title, body, headers, paragraphs, etc.
  5. Save the file and close.
  6. Back in terminal, paste python -m SimpleHTTPServer to start the development server.
  7. In the address bar (in Chrome), type localhost:8000/(name of html file you created).html
